# Fuentes de Datos y Atribuciones

## Dataset Principal

Los datos de la encuesta (`datos_encuesta_2026.csv`) y los instrumentos de recolección de información (cuestionarios en inglés y español) utilizados en este análisis provienen del siguiente dataset de Zenodo:

**Cabañas, M., Gómez García, L. D., Henao, L. C., Lis-Gutiérrez, J.-P., & Pareja Daza, C. P. (2026). Comparing emotional competencies and financial knowledge in mobilizing entrepreneurial action (Version V1) [Data set]. Zenodo. https://doi.org/10.5281/zenodo.18546128**

### Componentes del Dataset:
- **datos_encuesta_2026.csv**: Tabla de datos recolectados del dataset Zenodo
- **instrumentos/English Questionnaire.pdf**: Instrumento de recolección en inglés (Zenodo)
- **instrumentos/Spanish Questionnaire.pdf**: Instrumento de recolección en español (Zenodo)

## Análisis Propio

El archivo **analisis_encuesta_2026.ipynb** es una construcción propia desarrollada por Jorge Ariel Loaiza para el desarrollo de su tesis doctoral sobre emprendimiento. Este notebook contiene análisis, visualizaciones y procedimientos estadísticos aplicados al dataset de Zenodo.
