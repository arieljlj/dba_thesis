# Analysis Scripts

Scripts para análisis exploratorio, estadístico y generación de visualizaciones.

## Scripts/Notebooks disponibles

- **survey_analysis_2026.ipynb**: Análisis de la encuesta 2026 (Jupyter Notebook)
- **macroeconomic_analysis.py**: Análisis de datos macroeconómicos

## Requisitos

- Python 3.8+
- Jupyter Notebook
- Librerías: pandas, numpy, matplotlib, seaborn, scipy

## Uso

```bash
# Para notebook
jupyter notebook analysis/survey_analysis_2026.ipynb

# Para scripts Python
python analysis/macroeconomic_analysis.py
```

## Salida

Los gráficos, tablas y resultados se guardan en:
- `../data/outputs/figures/`: Gráficos
- `../data/outputs/tables/`: Tablas
- `../data/outputs/statistics/`: Estadísticas descriptivas

## Resultados generados

- Análisis descriptivo de variables
- Gráficos exploratorios
- Correlaciones y relaciones
- Estadísticas de la encuesta
