# Datos

Esta carpeta almacena los datos de la investigación doctoral.

## Organización sugerida

```
data/
├── raw/           # Datos crudos originales (NO modificar)
├── procesados/    # Datos limpios y procesados
├── instrumentos/  # Encuestas y formularios en formato digital
└── muestras/      # Información sobre la muestra seleccionada
```

> ⚠️ No subir datos personales de participantes sin anonimizar previamente.
